This webpage was designed to be similar in look and functionality with the 
Find My iPhone app used for Apple devices.

-when users open the page, they are informed, inobtrusively, that the previous session
has expired in order to safeguard their privacy.
-the "Sign in..." button is non-functional until input is picked up in both
the "AppleID" and "Password" boxes read input
-once the button is activated, if the user clicks on it, they are once again informed
inobtrusively that their login was a "Success!"
